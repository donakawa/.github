<h2>"살까 말까 고민될 땐, 도나에게 물어봐!"</h2>

도나카와는 무분별한 충동구매를 막고, 건강한 소비 습관 형성을 돕는 스마트 쇼핑 가이드 서비스입니다.
단순 지출 기록을 넘어, <br> AI 소비 코치 <strong> 도나 </strong> 가 구매 전 단계부터 개입하여 당신의 소비 의사결정을 도와줍니다.
위시리스트에 담긴 상품을 두고 AI와 대화하며<br>
소비의 우선순위를 정하고, 목표 예산에 맞춘 맞춤형 리포트를 받아보세요. <br>
귀여운 햄스터 캐릭터 '도나'의 변화를 통해 쇼핑의 즐거움은 유지하면서도, 
감정에 치우치지 않는 현명한 소비 생활을 시작할 수 있습니다.
